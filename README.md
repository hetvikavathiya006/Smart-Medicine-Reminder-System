🩺 Smart Medicine Reminder System
The Smart Medicine Reminder System is a user-friendly desktop application developed using Python. It helps users efficiently manage their daily medication routine by allowing them to enter medicine details such as name, morning dose, morning time, night dose, and night time.
The application provides an intuitive graphical user interface that allows users to add and delete medicine schedules easily. All entered data is displayed in a structured table format, making it simple to track and manage medications. This system is especially useful for individuals who need to follow a strict medication schedule.
## Features
Add medicine with morning and night schedule
User-friendly GUI interface
Display data in table format
Delete selected medicine entries
Supports 24-hour time format
## Technologies Used
Python
Tkinter (GUI Library)
▶️ How to Run
Install Python on your system
Download or clone this repository
Open terminal / command prompt
Run the program:
python smart_medicine.py
